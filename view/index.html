Discord Emoji API Documentation

GET /api/emoji

Retrieve Emoji information


Query Parameters
Field Type Description
name string Name of the emoji. It doesn't have to be exact. If the emoji name contains the string specified here, it will show in the results.
category string
  • Acceptable Categories
  • all, activities, flags, food, nature, objects, people, symbols, travel
    		•

    Example
    GET /api/emoji?name=heart&category=people

    Usage

    To use, we have to style an html (preferably a div) element/tag with background-image, background-size, and background-position 

    
  <div style="
    background-image: url('https://discord.com/assets/7b423a7d402beedfecb0bb81f9704953.png');
    background-position: -512px -96px;
    background-size: 1344px 1152px;
    width: 32px;
    height: 32px;"></div>
